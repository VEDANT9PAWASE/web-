
CB Invest - Full Version Website (v1.0)
--------------------------------------

Features Included:
- Homepage with styled investment plan
- User Registration/Login system (PHP + MySQL ready)
- User Dashboard: Track investment, earnings, referrals
- Daily ROI logic (1% until double)
- Referral system with unique links and earnings
- Admin panel: manage users, investments, referrals
- Responsive design with images/icons

Setup:
1. Upload all files to your hosting server (public_html)
2. Create a MySQL database and import 'database.sql'
3. Edit config.php to set your DB credentials
4. Access your domain to test the app

Default admin login: admin@example.com / admin123

For support, contact your developer.
